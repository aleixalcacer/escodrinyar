# Installation

Escodrinyar is a Python package that can be installed from its source code.
Below are the steps to install it:

## Installation from Source Code (Github Repository)

To install Escodrinyar from its source code, you need to clone its Github repository and run the installation command. 
Follow the steps below:

```shell
python -m pip install git+https://github.com/aleixalcacer/escodrinyar.git
```

This will install Escodrinyar on your system.
